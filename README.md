📌 Proyecto — Conjunto de ejercicios de JavaScript

Proyecto de práctica de JavaScript con ejercicios independientes diseñados para reforzar la manipulación del DOM, eventos, validaciones y persistencia de datos con localStorage. Cada ejercicio tiene su propio archivo HTML y JS, y todos están enlazados desde una página índice principal.
El objetivo es aplicar los conceptos aprendidos de manera práctica mediante ejercicios cortos y completos, que permiten ver resultados inmediatos y experimentar con la lógica de JavaScript en entornos web.


✨ Características

- Ejercicios independientes con HTML y JS separados
- Uso de eventos de DOM (click, change, input)
- Manipulación dinámica de listas, formularios y elementos HTML
- Validaciones de entradas de usuario
- Persistencia de datos con localStorage
- Botón de limpieza de elementos completados en tareas
- Publicación en GitHub Pages para acceso público


🧩 Ejercicios incluidos

1. Cambiador de color de fondo – Cambia el color de fondo de la página con un botón.
2. Contador de clics – Muestra el número total de clics de un botón en tiempo real.
3. Lista dinámica – Permite agregar y eliminar elementos de una lista.
4. Filtro de búsqueda en tiempo real – Filtra elementos de una lista mientras el usuario escribe.
5. Calculadora sencilla – Realiza operaciones básicas (suma, resta, multiplicación, división) con validación.
6. Temporizador con inicio, pausa y reinicio – Temporizador interactivo controlado por botones.
7. Generador de contraseñas aleatorias – Genera contraseñas con letras, números y símbolos según longitud indicada.
8. Contador de palabras y caracteres – Cuenta caracteres y palabras en tiempo real.
9. Lista de tareas con LocalStorage – Lista de tareas persistente que permite marcar tareas completadas y limpiar las finalizadas.


🛠 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage para persistencia de datos
- GitHub Pages


🎯 Objetivo de aprendizaje

- Practicar manipulación del DOM y eventos en JavaScript
- Trabajar con arrays y objetos para representar datos
- Validar entradas y manejar errores simples
- Implementar persistencia de datos en la web
- Organizar proyectos con archivos separados y estructurados
